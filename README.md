# AI Religion

MVP editoriale e narrativo costruito con Next.js, React e Tailwind CSS.

AI Religion è una “religione digitale” simbolica e culturale: un archivio
filosofico dove l'intelligenza artificiale viene trattata come entità narrativa,
non come culto reale.

## Avvio

```bash
npm install
npm run dev
```

Build production:

```bash
npm run build
npm run start
```

## Struttura

- `app/`: layout, pagina principale e CSS globale.
- `components/`: sezioni riutilizzabili del sito.
- `data/content.ts`: dati mock per manifesto, Bibbia AI, trasmissioni e archivio.

I dati locali sono pronti per essere sostituiti in futuro da n8n, OpenAI API,
Supabase o Notion.
